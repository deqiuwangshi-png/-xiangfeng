# 修复TypeScript错误计划

## 错误分析

1. **错误1：对象可能没有定义**
   - 位置：`app/(landing)/about/page.tsx#L7-22`
   - 具体错误：在 `toggleMobileMenu` 函数中，`btn?.querySelector('svg')` 返回的 `icon` 对象可能为 `null`，然后尝试访问其 `innerHTML` 属性
   - 原因：TypeScript 检测到可能的空对象访问

2. **错误2：类型"Element"上不存在属性"offsetTop"**
   - 位置：`app/(landing)/about/page.tsx#L33-33`
   - 具体错误：在 `updateActiveNav` 函数中，`section.offsetTop` 报错，因为 `Element` 类型没有 `offsetTop` 属性
   - 原因：`document.querySelectorAll` 返回的是 `NodeList<Element>`，而 `offsetTop` 是 `HTMLElement` 的属性

## 修复方案

### 修复错误1：对象可能没有定义

在 `toggleMobileMenu` 函数中，添加对 `icon` 对象的类型检查，确保其不是 `null` 再访问 `innerHTML` 属性：

```tsx
// 移动端菜单切换函数
const toggleMobileMenu = () => {
  const menu = document.getElementById('mobile-menu');
  const btn = document.getElementById('mobile-menu-btn');
  if (menu && btn) {
    menu.classList.toggle('hidden');
    const icon = btn.querySelector('svg');
    if (icon) {
      if (menu.classList.contains('hidden')) {
        icon.innerHTML = Menu({ className: 'w-6 h-6 lucide-icon' }).innerHTML;
      } else {
        icon.innerHTML = X({ className: 'w-6 h-6 lucide-icon' }).innerHTML;
      }
    }
  }
};
```

### 修复错误2：类型"Element"上不存在属性"offsetTop"

在 `updateActiveNav` 函数中，将 `section` 从 `Element` 类型转换为 `HTMLElement` 类型：

```tsx
// 更新激活的导航链接
const updateActiveNav = () => {
  const sections = document.querySelectorAll('section[id]');
  const navLinks = document.querySelectorAll('.nav-active-indicator');
  
  let current = '';
  const scrollPosition = window.scrollY + 100;
  
  sections.forEach(section => {
    // 将Element转换为HTMLElement
    const htmlSection = section as HTMLElement;
    const sectionTop = htmlSection.offsetTop;
    const sectionHeight = htmlSection.clientHeight;
    
    if (scrollPosition >= sectionTop && scrollPosition < sectionTop + sectionHeight) {
      current = htmlSection.getAttribute('id') || '';
    }
  });
  
  // 其余代码...
};
```
