# 重构About页面组件计划

## 问题分析

当前`about`页面组件存在多个问题，包括：

1. **直接DOM操作**：使用`classList.toggle()`、`innerHTML`等直接操作DOM
2. **事件监听器泄漏**：在`useEffect`中添加的事件监听器没有正确清理
3. **状态管理问题**：使用DOM操作代替React状态管理
4. **类型安全问题**：缺乏适当的类型守卫和类型检查
5. **性能问题**：在事件处理器中创建新函数，没有使用`useCallback`
6. **可访问性问题**：缺乏适当的ARIA属性和键盘导航支持

## 重构方案

### 1. 组件状态管理

- 使用`useState`管理移动端菜单的显示/隐藏状态
- 使用`useRef`引用DOM元素，避免直接DOM操作
- 使用条件渲染代替`classList.toggle()`

### 2. 事件处理

- 使用React合成事件系统，避免直接DOM事件绑定
- 在`useEffect`中正确清理所有事件监听器
- 使用事件委托，减少事件监听器数量
- 使用`useCallback`包装事件处理函数，避免不必要的重渲染

### 3. TypeScript类型安全

- 添加适当的类型守卫
- 使用类型断言时确保元素存在
- 为所有函数和变量添加明确的类型

### 4. 可访问性增强

- 为按钮添加`aria-label`属性
- 确保键盘导航支持
- 添加焦点管理
- 确保所有交互元素都有适当的ARIA属性

### 5. 性能优化

- 使用`useCallback`包装函数，避免在render中创建新函数
- 使用适当的依赖数组，避免不必要的`useEffect`触发
- 减少DOM操作，使用React虚拟DOM

## 重构实现步骤

### 步骤1：添加状态管理

```tsx
// 添加状态管理
const [mobileMenuOpen, setMobileMenuOpen] = useState(false);
const mobileMenuRef = useRef<HTMLDivElement>(null);
const mobileMenuBtnRef = useRef<HTMLButtonElement>(null);
```

### 步骤2：重写toggleMobileMenu函数

```tsx
// 使用useCallback包装函数
const toggleMobileMenu = useCallback(() => {
  setMobileMenuOpen(prev => !prev);
}, []);
```

### 步骤3：重构导航激活逻辑

```tsx
// 重写导航激活逻辑，使用useState和useEffect
const [activeSection, setActiveSection] = useState('');
const sectionsRef = useRef<HTMLElement[]>([]);

// 使用useCallback包装函数
const updateActiveNav = useCallback(() => {
  const scrollPosition = window.scrollY + 100;
  
  sectionsRef.current.forEach(section => {
    if (!section) return;
    
    const sectionTop = section.offsetTop;
    const sectionHeight = section.clientHeight;
    
    if (scrollPosition >= sectionTop && scrollPosition < sectionTop + sectionHeight) {
      setActiveSection(section.id);
    }
  });
}, []);
```

### 步骤4：修复useEffect事件监听器

```tsx
// 修复useEffect，正确清理事件监听器
useEffect(() => {
  // 添加滚动事件监听
  window.addEventListener('scroll', updateActiveNav);
  
  // 初始更新一次导航激活状态
  updateActiveNav();
  
  // 点击外部关闭移动端菜单
  const handleClickOutside = (event: MouseEvent) => {
    if (mobileMenuRef.current && mobileMenuBtnRef.current &&
        !mobileMenuRef.current.contains(event.target as Node) &&
        !mobileMenuBtnRef.current.contains(event.target as Node)) {
      setMobileMenuOpen(false);
    }
  };
  
  // 添加点击外部事件监听
  document.addEventListener('mousedown', handleClickOutside);
  
  return () => {
    // 清理所有事件监听器
    window.removeEventListener('scroll', updateActiveNav);
    document.removeEventListener('mousedown', handleClickOutside);
  };
}, [updateActiveNav]);
```

```tsx
// 添加aria-label到按钮
```

```tsx
  return element !== null && element !== undefined && 'offsetTop' in element;
```
