# -开发历程
本次项目采用Next+supabase为主，由AI帮我构建完善基本的框架页面，此仓库包含基本的技能、项目规则和相关文档，，耗时一周
宏观上这个相逢应该是从25年11月开始构建，每月一次就会重构，原因AI给出的代码埋雷太多了，容易崩溃，一直持续到26年2月份才会逐渐改善
在此期间，共计重构5次框架页面，第6次完善基本的页面，后续打算慢慢构建
